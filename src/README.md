# MNIST Neural Network from Scratch (numpy)

A simple fully connected neural network trained on the MNIST using only NumPy and manual forward pass + backpropagation.

## Data
This project expects the MNIST CSV files from:
https://www.kaggle.com/datasets/oddrationale/mnist-in-csv

Place the following files in a `data/` directory:
- mnist_train.csv
- mnist_test.csv
